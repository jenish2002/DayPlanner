# 📅 Day Planner

A productivity-focused Android application designed to help users manage their daily routine and stay on track. Day Planner reminds you of each task at your specified time — so you never have to remember anything on your own.

---

## ✨ Features

- **Add Daily Tasks** — Create and organize your work or daily routine tasks
- **Set Reminders** — Get notified at the exact time you set for each task
- **Task Management** — View, edit, and delete tasks easily
- **Clean UI** — Simple and intuitive interface for a smooth user experience

---

## 🛠️ Tech Stack

- **Language:** Java, XML
- **IDE:** Android Studio
- **Platform:** Android
